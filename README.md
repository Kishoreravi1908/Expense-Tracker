## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No installation required!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Kishoreravi1908/Expense-Tracker.git
```

2. Navigate to the project directory:
```bash
cd Expense-Tracker
```

3. Open `index.html` in your web browser:
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

## Usage

1. **Add Expense**: Click the "Add Expense" button and fill in the expense details
2. **Set Budget**: Define your budget limits for different spending categories
3. **View Summary**: Check your spending patterns and remaining budget
4. **Get Alerts**: Receive notifications when approaching or exceeding budget limits
